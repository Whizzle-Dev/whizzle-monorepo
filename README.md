## Technologies Used:

* Language: **Typescript**
* Framework: **Next.js, NestJS**
* Query builder: **Kysely**
* Migrations manager: **Prisma**
* Database: **Postgres**
* Queues/Jobs: **Bull/Redis**
* Package manager: **PNPM**
* Emails: **Resend**
* Error Reporting: **Sentry**
* Infrastructure: **Google Cloud, Terraform, Upstash**
