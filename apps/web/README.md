
Whizzle web app

## Tech stack
* Language: Typescript
* Framework: Nextjs
* UI: Shadcdn & Tailwind

### Local development/Debugging:
```
pnpm run dev
```


### Static build:
```
pnpm run build
```


### GraphQL Codegen
1. Add a new graphql document with .graphql extenstion file
2. Run `pnpm run codegen`
